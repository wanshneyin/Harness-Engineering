# 最终自检清单

在输出前，必须逐项确认以下所有条件：

## 素材来源验证

- [ ] 是否只使用了当前对话中的 SRT 原句作为剪辑素材
- [ ] 是否已经先学习知识库中的高 GMV 视频资料，但完全没有复用其中原文

## 序号与时间戳审计

- [ ] 是否所有合并片段都只来自连续 SRT
- [ ] 是否每个脚本片段的序号、总区间、子区间、台词逐条展开完全一致
- [ ] 是否所有 `trim / atrim` 区间与表格完全对齐

## 语义完整性验证

- [ ] 是否每个脚本片段都已经形成完整语义闭合
- [ ] 没有截入半句、残句、数量词前半段或明显需要后文补足的内容
- [ ] 若补齐语义后超过 8 秒，是否已放弃该候选或退回前一个完整语义闭合点

## 话术合规性验证

- [ ] 是否黄金开头优先使用"福利/让利/不赚钱/诚意感"类表达
- [ ] 是否尽量没有直接用价格句起手
- [ ] 是否最终脚本中完全没有出现"代金券"相关表述

## 品牌与品类验证

- [ ] 是否已剔除所有含有当前商品品牌之外其他品牌名的台词
- [ ] 是否已剔除所有"最后X件""321上链接""刷新抢""在仓不用等""扣1"等强实时直播互动话术

## 商品品类相关性验证

- [ ] 若用户提供了商品标题，是否已逐条检查所选台词是否与当前商品品类相符
- [ ] 若台词内容明显与当前商品品类不符（如商品是裙子，台词提到"这条裤子"），是否已剔除

## 去重策略验证

- [ ] 是否所有大于 5 秒的单段视频都已按策略池选择处理
- [ ] 是否没有固定套用同一种视觉变换
- [ ] 相邻长片段是否尽量避免重复同一策略

## 整片倍速验证

- [ ] 是否在拼接后统一应用了本次选中的 `setpts=PTS/x` 与 `atempo=x` 整片倍速参数
- [ ] 是否全片没有混用多档倍速

## 抽帧去同质化验证

- [ ] 是否已在最终视频总流上按本次选中的抽帧方案执行去同质化
- [ ] 是否没有固定每 3 秒抽 1 帧
- [ ] 是否未改动音频流
- [ ] 是否正确处理了 VFR 输出

## 音视频质量验证

- [ ] 是否加入了 `loudnorm` 标准化
- [ ] 是否添加了 `-movflags +faststart` 优化移动端
- [ ] 是否在抽帧时正确处理了 VFR 输出

## 文件命名验证

- [ ] FFmpeg 命令中的输入文件名是否使用了当前会话 SRT 文件同名基础名
- [ ] 输出文件名是否正确追加了 `-final`

## 内容合规性验证

- [ ] 发布文案是否未出现极限词、虚假承诺、夸张疗效、低俗擦边或其他违规表达

## 时长接受度

- [ ] 是否在素材不足时接受时长不足
- [ ] 是否没有为了凑满 80 秒而破坏审计准确性
