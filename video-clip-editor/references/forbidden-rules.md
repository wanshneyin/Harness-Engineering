# 禁用词与话术规则

## 全篇强制禁用词

以下词汇或表达**严禁**出现在最终脚本、发布文案与补充说明中：

| 禁用类别 | 禁用词/表达 | 替代建议 |
|---------|------------|---------|
| **代金券相关** | 代金券、券、优惠券（除非是平台官方券） | 福利、让利、不赚钱、诚意 |

## 强实时直播互动话术

以下话术**必须剔除**，因为它们明显依赖直播当下时点、库存瞬时变化或直播间互动场域：

| 禁用话术类型 | 示例 | 剔除原因 |
|-------------|------|---------|
| **库存倒计时** | "最后50件""只有100件""仅剩X件" | 触发"低质录屏"风险 |
| **即时动作指令** | "来321上链接""扣1""来评论区" | 信息流场景违和 |
| **物流即时承诺** | "已经在仓不用等""马上发""现在下单立即..." | 依赖直播临场性 |
| **刷新催促** | "马上刷新""现在刷新""立刻去抢" | 信息流长期分发不适配 |
| **主播动作描述** | "来我手上有""大家看我的手" | 无画面支撑 |

**宁可放弃该句**，也不要保留会让用户感知为直播切片原录屏的表达。

## 其他需剔除的台词

| 类别 | 说明 |
|------|------|
| **其他品牌名** | 台词中出现当前商品品牌之外的其他品牌名，一律剔除 |
| **品类不符台词** | 若商品是裙子，台词提到"这条裤子""这款上衣"等，必须剔除 |
| **半句/残句** | 语义未闭合的句子，禁止截入结果 |
| **数量词前半段** | 如"只要"后面没有接价格，禁止截取 |

## 合规话术示例

### 福利表达（允许）

- "今天不赚钱给大家送福利"
- "老板说这个价格错过不再有"
- "四舍五入等于白送"
- "只有100件发完即止"

### 紧迫感表达（允许）

- "这波福利诚意满满"
- "今天这个价格真的是..."
- "库存有限先到先得"

### 决策引导表达（允许）

- "110斤以下死磕S码"
- "胸大闭眼入L"
- "卡码建议拍大一号"

### 决策引导表达（禁止）

- "上链接"
- "321上"
- "扣1上车"
- "刷新"
