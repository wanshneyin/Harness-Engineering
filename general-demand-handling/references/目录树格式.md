```text
接口与数据源（互通相关）
│
├─ PHP：@/phpApi/groupChat.js
│  ├─ getShowGroupList()                          # 【已有】返回群列表，需后端新增字段：

│  ├─ getInteropSwitch()                          # 【新增】总开关接口，返回：
│  │                                              #   - interopEnabled: 0/1（总开关状态）

│  └─ getGroupDetail()                            # 【已有】群详情，需后端新增：isInteropGroup 字段
│
├─ Java：@/javaApi/groupChat.js
│  └─ (暂无新增，卡片翻译仍走现有 translateText 接口)
│
├─ NIM SDK（云信）
│  ├─ msg.custom                                  # 消息扩展字段，后端写入：
│  │                                              #   - senderRegion: 'domestic'/'international'（发送者区域）
│  │                                              #   - translateText: '...'（中文译文，国际→国内时后端生成）
│  │                                              #   - hasTranslation: 0/1（是否有译文）
```
